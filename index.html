Compare viewports

This script records the last viewport's width value and the newest one when the resize event is fired and finally compares the two values, so then you could make an action if it gets wider otherwise it will trigger another function.

Please, take this little demo as it is, a little demo where I pretend to show to you the power of this plugin, but consider your own target before copy/paste. It is possible that you need to reset some properties contained in the CSS file.
